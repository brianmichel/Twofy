The missing 2FA helper application.

Getting two factor codes from your Messages app into your browser is pain and there is no API to allow browser developers to easily get these codes. Twofy is an open sourced, sandboxed application that helps automate this process without any hassle.

Why?

This idea isn't new, but it a continuation of the same ideas that have been used in Alfred, Raycast, and countless other helper applications in the open source world. I wanted something that would work in all of the browsers that I use. I also wanted the application to be sandboxed and easily verifiable that no networking was taking place to ensure my codes were being securely handled.

It's important to me that something that can read your text messages is open source which allows anyone to inspect the code running on their machine and make an informed decision about that piece of software.

How It Works

When Twofy prompts you for the Messages directory this grants the application access to read the database where all of the messages you receive live on disk. The application can then set up a polling listener on that database to find codes that match a particular regex.

If the query finds anything it dispatches an encrypted distributed notification that can only be decrypted by the helper application that the browser extension communicates with. Once that browser support application receives that notification, it decrypts it and sends it into the service worker within the browser extension process.

Features

Twofy is simple, but feature rich to help make your workflows a bit more straightforward when needing just-in-time access to your two factor authentication codes that have been sent via text message.

  • Sandboxed
    You must give explicitly grant access to your message data in order for Twofy to do anything. It can't do anything automatically with your data.
  • Notarized
    Twofy is Notarized by Apple to ensure that the binary has not been tampered with. This ensures your version of Twofy is coming from my developer id.
  • Encrypted IPC
    Twofy's processes use encrypted messages to securely pass codes between processes.
  • Open Source
    The main application, helper application, extensions, and even this site are all open source on GitHub to allow anyone to inspect their contents.
